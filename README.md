<h1 align="center">Module</h1>

### log

* 日志工具包，可用于简单的日志输出。

* 日志分为5个等级：`DEBUG`，`INFO`，`WARN`，`ERROR`，`FATAL`
  
* Maven依赖

    ```xml
    <dependency>
      <groupId>io.github.jxnflzc.jtools</groupId>
      <artifactId>log</artifactId>
      <version>${jtools.version}</version>
    </dependency>
    ```
